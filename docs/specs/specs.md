# [MAJOR COURSE OUTPUT 1] MONTE CARLO AND STATISTICAL GAME SIMULATION

This project requires you to design and develop a modular simulation environment that demonstrates games of chance into a statistical laboratory, allowing users to stress-test the game of choice against the concept of the Law of Large Numbers. This R **Shiny App module** abstracts the physical components of gaming into an engine to study the theoretical (a priori) and frequentist view (a posteriori) probabilities of success.

1. The first phase of this project involves selecting a **specific game of chance and determining a single outcome** to serve as the focus of this statistical inquiry. Provide a mathematical derivation of the theoretical probability of success for the outcome of choice by defining the precise ratio between favorable outcomes and the total sample space.
2. Once the game parameters are set, implement a **simulation layer that generates a convergence trace to visualize the Law of Large Numbers in action**. As the number of trials increases, the engine calculates the running proportion of successes, or the frequentist estimated probability, and plots it against the fixed theoretical line. The engine must employ Maximum Likelihood Estimation (MLE) to derive the most probable value for the probability of success based on the observed data.
3. Develop and compare **multiple (frequentist-based) Confidence Intervals to quantify the uncertainty of the probability estimate**. Visualize the differences between the Standard (Wald) Score, the Wilson Score, and the Agresti-Coull intervals. Plotting these intervals side-by-side to see which method offers the most robust "safety net" for the estimated success rate. Also, create a visualization that shows the behaviors of these confidence intervals as the sample size grows.

As part of your submission, please prepare a **technical documentation for both the R Shiny application structure and the underlying statistical scripts**. Detailed documentation should clearly delineate the reactive logic within the server function and provide annotated explanations for the mathematical functions used in the estimation of confidence intervals. This should be provided in .pdf format, together with the R Shiny App script in .R that you will submit.
